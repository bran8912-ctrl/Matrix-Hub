# Casino Game Math

## House Edge
15% structural [edge](https://en.wikipedia.org/wiki/Casino_game#House_advantage).

### Allocation
- 10% → [Liquidity](https://en.wikipedia.org/wiki/Market_liquidity)
- 3% → Casino Reserve
- 2% → Development

## Expected Value
[EV](https://en.wikipedia.org/wiki/Expected_value) = -0.15 across all games.
Variance changes, math does not.
