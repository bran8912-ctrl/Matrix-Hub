# MTX Casino Games

## Dice Roll
Roll under target to win.

## Coin Flip
50/50 outcome.

## Number Match
Pick 1–100. Exact match wins.

## Slot Probability
Weighted symbols, fixed [RTP](https://en.wikipedia.org/wiki/Return_to_player).

## Crash (Phase 2)
Multiplier grows until crash.

## PvP Arena (Phase 3)
Player vs Player wagers.
