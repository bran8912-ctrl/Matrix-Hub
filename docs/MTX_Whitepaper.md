# Matrix-HubCoin (MTX) Whitepaper

MTX is a [utility token](https://en.wikipedia.org/wiki/Utility_token) powering the Matrix-Hub ecosystem.
It is designed for circulation, access, and sustainability —
not speculation.

## Core Functions
- Access to Matrix-Hub apps
- [Casino game](MTX_Casino_Games.md) participation
- [Liquidity](https://en.wikipedia.org/wiki/Market_liquidity) reinforcement
- [Governance](MTX_DAO_Governance.md) signaling

MTX has no profit guarantees, no dividends, and no ownership claims.
